FAQ
  • ¿Qué es HTML? HTML es el lenguaje de marcado estándar para crear páginas web.
  • ¿Qué significa CSS? CSS significa Hojas de Estilo en Cascada y se utiliza para estilizar el contenido HTML.
  • ¿Qué es JavaScript? JavaScript es un lenguaje de programación que se utiliza para crear interactividad en páginas web.
